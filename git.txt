git clone url: clone a respiratory from GitHub to the local computer
git add file: Add a new file
git commit -m "message": Commit creating a new file
git commit -am "message": Commit update changes to all the file
git push: Update the file in GitHub correspond to the local computer
git pull: Update the file in the local computer the same as in GitHub
git log: see all the commits so far
q: exit any git command
git status: status if the local file has been committed
git reset --hard <commit>: go back to a commit
git reset --hard origin/master: change to same as in GitHub

When decide to make a new feature, create a new change
git branch: show all the branches, * denotes current branch
git checkout -b <name>: create a new branch call <name>
git checkout <name>: switch to the <name> branch
git merge <name>: merge current brach with <name> branch
